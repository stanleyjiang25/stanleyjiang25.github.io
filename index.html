The Whether Bee - Instant Messaging Web Application

The Whether Bee is an instant chat app with a twist. In addition to many of the features of normal chat rooms (such as exchanging messages or creating group chats), users will be able to talk with chat bots, access certain commands, and enjoy our beautiful user interface.

Primary Features
  • Contacts List - add other users (and bots) to a contacts list
  • Conversations - create conversations with contacts
  • User Registration - elegant user registration interface
  • Chat bots - chat bots to talk to and include in conversations
Implementation
  • Used OCaml (including Opium and Caqti third-party libraries) to create endpoints on our back-end system and support other back-end functionality
  • Efficiently store and query user data in a database with PostgreSQL
  • Elegant graphical front end in a web browser built using Javascript and VueJS

Trainstris - Tetris Training Platform

For Tetris beginners trying to improve their game, Trainstris offers a series of carefully curated levels designed to teach players how to stack quicker and evaluate their board state. With our many in-game scenarios and revolutionary scoring methodology, Trainstris gives meaning to mindless practice and will hopefully help you achieve your PPS (Pieces Per Second) aspirations!

Primary Features
  • Lessons - 15 different levels teaching various Tetris fundamentals
  • Evaluation - board state evaluation to measure improvement
  • Sandbox Mode - experiment different ways to play
  • Interface - easy to use interface
Implementation
  • Used Javascript to build game components and support evaluation
  • Browser-based user interface built using Javascript and ReactJS
